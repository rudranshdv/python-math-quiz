# Python Math Quiz

A beginner friendly python project that generates a random addition problem and allows the user to answer it

## Features
- Generates two random numbers between 1 and 10
- Asks the user to solve an addition problem
- Displays the correct answer
- Displays the user's answer

## Concepts Used
- Python modules (`random`)
- Random number generation
- Variables
- User input
- Integer data type
- Arithmetic operations
- Type conversion (`int()`)

## Example Output

```text
What is 4 + 7? 11

Correct answer was: 11
Your answer was: 11
```

Another Example:

```text
What is 3 + 9? 10

Correct answer was: 12
Your answer was: 10
```
